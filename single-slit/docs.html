Single-Slit Diffraction

When coherent light passes through a single narrow slit, it spreads and produces a characteristic diffraction pattern: a bright central maximum flanked by progressively weaker secondary maxima, separated by sharp dark minima.

This is Fraunhofer (far-field) diffraction — the regime where the observation screen is far enough from the slit that the wavefronts arriving at any point can be treated as plane waves.

Historical Background

1665
Francesco Grimaldi first documented diffraction of light around obstacles, naming the phenomenon.
1678
Christiaan Huygens proposed that every point on a wavefront acts as a secondary source — the foundation of diffraction theory.
1801
Thomas Young's double-slit experiment demonstrated the wave nature of light.
1816
Augustin-Jean Fresnel combined Huygens' principle with wave interference, deriving the sinc² intensity formula rigorously.
1882
Joseph von Fraunhofer developed the far-field approximation, enabling the clean closed-form formula used today.

Key parameters:

  • a — slit width (determines spread of diffraction)
  • λ — wavelength of light
  • L — distance from slit to screen (Fraunhofer regime: L ≫ a²/λ)

Derivation

Divide the slit of width a into infinitesimal Huygens sources. Each element dy at position y₀ within the slit contributes a wave to point P on the screen.

For a screen point at angle θ, the path difference between a source at the top of the slit vs. the bottom is:

Δtotal = a sin θ

Integrating all contributions from −a/2 to +a/2 (treating each as a phasor rotating at rate k sin θ):

E(θ) = E₀ · ∫−a/2+a/2 eikx sinθ dx = E₀ · a · sinc(πa sinθ / λ)

The intensity is the square of the field amplitude:

I(θ) = I₀ · sinc²(πa sinθ / λ)

Intensity Formula

I(y) = A² · sinc²(β)
where β = πa sinθ / λ, sinθ = y / √(y² + L²)

This simulation uses the exact geometry — no small-angle approximation. The sinc function is defined as:

sinc(x) = sin(πx) / (πx), sinc(0) = 1

Note: at β = 0 (center), sinc(0) = 1, so I = A² (maximum). Zeros occur whenever sin(β) = 0 and β ≠ 0, i.e., β = ±1, ±2, ...

Minima & Secondary Maxima

Dark Minima

a sin θm = mλ, m = ±1, ±2, ±3, …

Screen positions (small-angle approximation):

ym ≈ mλL / a

Central Maximum

Width of the central maximum (distance between first minima on each side):

Wcentral = 2λL / a

Secondary Maxima

Approximate positions (midway between minima):

a sin θ ≈ (m + ½)λ, m = ±1, ±2, …

Intensities of secondary maxima: ~4.7%, 1.7%, 0.83%, … of the central peak.

Guided Experiments

1. Diffraction vs. Slit Width

Set λ = 550 nm, L = 1 m. Vary slit width from 1.0 mm down to 0.05 mm. Observe how the pattern widens dramatically as a decreases. Verify: central max half-width y₁ = λL/a.

2. Wavelength Dependence

Fix a = 0.2 mm, L = 1 m. Sweep wavelength from 380 nm (violet) to 750 nm (red). Red light diffracts more than violet — the pattern widens with λ.

3. Geometric Optics Limit

Set λ = 550 nm. Increase slit width to 1.0 mm. Note the pattern nearly collapses to a single bright bar — the geometric optics regime where a ≫ λ.

4. Secondary Maximum Intensities

Set a = 0.1 mm, L = 1.5 m. Look at the first secondary maximum — it should be about 4.7% as bright as the central peak. Toggle minima labels to identify positions.

5. Far-Field Condition

The Fraunhofer approximation requires L ≫ a²/λ. For a = 0.5 mm, λ = 550 nm: minimum L = (0.5e-3)²/550e-9 ≈ 0.45 m. Try L = 0.1 m and 2.0 m and compare.

6. Missing Orders Connection

Compare with the double-slit simulator. The sinc² envelope you see here is the same envelope that modulates double-slit fringes. When a = d/N, the Nth double-slit order falls exactly on a single-slit minimum — the "missing order."
