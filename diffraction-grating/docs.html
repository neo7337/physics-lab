Diffraction Grating

A diffraction grating consists of many equally-spaced slits. The interference of N coherent sources produces extremely sharp, bright principal maxima — making gratings the most powerful tool for separating wavelengths in spectroscopy.

Unlike a double-slit (which gives broad fringes), a grating with many slits concentrates energy into narrow peaks, dramatically improving the ability to resolve closely-spaced wavelengths.

Historical Background

1785
David Rittenhouse constructed the first diffraction grating from fine hairs strung between two parallel screws.
1813
Joseph von Fraunhofer independently invented the grating and used it to catalogue dark absorption lines in sunlight — now called Fraunhofer lines.
1882
Henry Rowland invented the ruling engine for precision concave gratings, enabling accurate atomic spectroscopy.
1960s
Holographic gratings produced by laser interference replaced ruled gratings for many applications, offering superior regularity.
Today
Echelle gratings achieve resolving powers >10⁵ in astronomical spectrographs. VPH (volume phase holographic) gratings approach 95% diffraction efficiency.

Key parameters: grating period d (slit-to-slit distance), slit width a, number of slits N.

Derivation

Each slit contributes a field with phase relative to the first slit of:

φₙ = 2πnd sinθ / λ = 2nδ

Summing N phasors with equal spacing δ = πd sinθ/λ:

E(θ) = E_single · Σₙ₌₀^{N-1} e^{i2nδ} = E_single · sin(Nδ) / sin(δ) · e^{i(N-1)δ}

The intensity includes the single-slit envelope (sinc²) and the N-slit interference factor:

I(θ) = I₀ · sinc²(πa sinθ/λ) · [sin(Nπd sinθ/λ) / (N sin(πd sinθ/λ))]²

Intensity Formula

I = A² sinc²(β) · G²(δ)
β = πa sinθ/λ  |  δ = πd sinθ/λ  |  G(δ) = sin(Nδ) / (N sinδ)

Principal Maxima

Occur when sin(δ)/sin(δ) = 1, i.e., when δ = mπ:

d sinθ = mλ   (m = 0, ±1, ±2, …)

Secondary Maxima

Between principal maxima there are N−2 weak secondary maxima. Their peak intensity is approximately 1/(N²) times the principal maximum (for large N, effectively zero).

Principal Maxima Width

Half-width of each principal maximum: Δ(sinθ) = λ/(Nd). Larger N → narrower, sharper peaks.

Resolving Power

The Rayleigh criterion gives the minimum resolvable wavelength difference:

R = λ/Δλ = mN

For a grating with N = 6 slits in first order (m = 1), R = 6. For a spectrograph grating with N = 50,000 slits and m = 2: R = 100,000 — enough to resolve sodium D lines separated by 0.6 nm.

Missing Orders

If the grating period d is an integer multiple of the slit width a, certain principal maxima coincide with zeros of the single-slit envelope — those orders are absent from the pattern:

Missing order condition: d/a = integer M → order m = M is missing

For example, with d = 1.0 mm and a = 0.20 mm: d/a = 5, so the m = ±5 order is suppressed.

Try this in the simulator: set a = 0.20 mm, d = 1.0 mm (d/a = 5) and observe the missing 5th-order peak.

Guided Experiments

1. Effect of Slit Number

Set N=2, observe broad fringes. Increase N to 6, then 12, then 20. Watch the fringes sharpen into narrow spikes as N increases — this is the grating's resolving power at work.

2. Missing Orders

Set d = 1.0 mm, a = 0.20 mm (d/a = 5). Identify the missing 5th-order peak. Then try d = 0.4 mm, a = 0.20 mm (d/a = 2) — the 2nd order disappears.

3. Wavelength Separation

Set N = 20. Note how the m = ±1 peaks for 400 nm (violet) vs 700 nm (red) are at very different positions. This is how a spectrometer separates spectral lines.

4. High vs. Low Dispersion

Compare d = 0.1 mm (high dispersion — orders widely spread) vs d = 5.0 mm (low dispersion — orders compressed near center). High-dispersion gratings are used for fine spectroscopy.

5. N-slit vs. Double-slit

Set N = 2 and compare with the double-slit simulator. They should produce identical patterns when a and d match. Increasing N narrows each fringe without changing the peak positions.

6. Angular Dispersion

Set λ = 550 nm, d = 0.5 mm. Record the m=+1 peak position. Now change λ to 580 nm. The shift Δy is proportional to Δλ·L·m/d — angular dispersion dθ/dλ = m/(d cosθ).
