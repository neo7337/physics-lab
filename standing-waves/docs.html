Standing Waves on a String

When two counter-propagating waves of equal frequency and amplitude travel along a stretched string, they superpose to form a standing wave — a pattern that oscillates in place, with fixed points of zero displacement (nodes) and points of maximum displacement (antinodes).

Standing waves arise naturally whenever a wave reflects off a boundary. On a string fixed at both ends, only certain discrete frequencies can sustain a standing wave — these are the resonant frequencies or harmonics of the string.

This simulation models the exact standing wave equation with tunable tension, linear density, string length, harmonic mode, and exponential damping.

Historical Background

~550 BC
Pythagoras observed that plucked strings produce harmonious tones when their lengths are in simple integer ratios — the first quantitative link between mathematics and music.
1636
Marin Mersenne published Harmonie Universelle, deriving what are now called Mersenne's Laws: string frequency is proportional to 1/L, √T, and 1/√μ.
1727
Johann Bernoulli and Brook Taylor derived the wave equation for a vibrating string from Newtonian mechanics, showing it admits sinusoidal solutions.
1753
Daniel Bernoulli proposed the superposition principle: any string vibration is a sum of its normal modes (harmonics). This was contested by Euler and d'Alembert for decades.
1787
Ernst Chladni visualized 2D standing wave patterns in thin plates using sand (now called Chladni figures), extending standing wave ideas to 2D surfaces.
1822
Joseph Fourier proved Bernoulli's superposition claim rigorously: any periodic function can be decomposed into a sum of harmonics — founding Fourier analysis.
1864
James Clerk Maxwell showed that electromagnetic waves also form standing waves, leading to the theory of cavity resonators and, eventually, quantum mechanics.

Key parameters:

  • L — string length (m); sets the allowed wavelengths
  • T — tension (N); determines the wave speed
  • μ — linear mass density (kg/m); also affects wave speed
  • n — harmonic mode number (1 = fundamental, 2 = first overtone, …)

The string is fixed at both ends (Dirichlet boundary conditions: y(0,t) = y(L,t) = 0), which forces the allowed wavelengths to be λn = 2L/n.

Derivation

Wave Equation

For a uniform string under tension T with linear density μ, Newton's second law for a small element gives the 1D wave equation:

∂²y/∂t² = (T/μ) · ∂²y/∂x² = v² · ∂²y/∂x²

where the wave speed is:

v = √(T / μ)

Boundary Conditions & Normal Modes

With both ends fixed (y = 0 at x = 0 and x = L), the general solution must satisfy:

y(0, t) = 0 and y(L, t) = 0 for all t

This restricts the allowed wave numbers to:

kn = nπ / L, n = 1, 2, 3, …

Corresponding wavelengths:

λn = 2L / n

Standing Wave Equation

A right-traveling wave A sin(knx − ωt) plus its left-traveling reflection A sin(knx + ωt) superpose to give:

y(x, t) = A · sin(nπx / L) · cos(ωt)
ω = 2πfn, fn = n · v / (2L)

This is a product of a space function sin(nπx/L) and a time function cos(ωt) — the defining property of a standing wave.

Mersenne's Laws

Combining fn = nv/(2L) and v = √(T/μ), the fundamental frequency is:

f₁ = (1 / 2L) √(T / μ)
Frequency ∝ 1/L, ∝ √T, ∝ 1/√μ

These are Mersenne's Laws — the three relationships known empirically since 1636, now derived from first principles.

Resonance

What is Resonance?

Resonance occurs when a driving force oscillates at exactly a natural frequency of the system. For a string fixed at both ends, the resonant frequencies are fn = nf₁ for integer n. At these frequencies, energy builds up efficiently in the standing wave pattern.

Quality Factor

A real string loses energy to air resistance and internal friction. The quality factor Q describes how sharply resonant the system is:

Q = fn / Δf

where Δf is the full-width at half-maximum of the resonance peak. High Q means narrow resonance and slow decay; low Q means broad resonance and rapid decay.

In this simulation, the Damping slider applies exponential amplitude decay e−γt, equivalent to a resistive damping force proportional to velocity.

Energy

The total mechanical energy stored in the nth mode is:

En = ½ μ L ωn² A²

This is equally split between kinetic energy (when string passes through equilibrium) and potential energy (when string is at maximum displacement), analogous to a simple harmonic oscillator.

Key insight: frequency depends only on v = √(T/μ) and L — not on amplitude. This is the principle behind tuning a string instrument: tighten (T↑) or shorten (L↓) to raise pitch.

Guided Experiments

1. Mersenne's Laws — Tension

Set n=1, μ=0.010 kg/m, L=1.0 m. Note the frequency at T=2 N. Increase T to 8 N (4× increase). Frequency should double (f ∝ √T). Verify: f should be √4 = 2× the original.

2. Mersenne's Laws — Length

Fix T=2 N, μ=0.010 kg/m, n=1. Record f at L=1.0 m. Halve the length to L=0.5 m. Frequency should exactly double (f ∝ 1/L). This is the physics behind placing a finger on a guitar string.

3. Counting Nodes

Step through modes n=1 to n=8. Count the red node dots at each mode — you should find exactly n+1 nodes (including both fixed endpoints) and n green antinode dots, confirming the theory.

4. Damping Decay

Set damping = 0.3, then watch the string amplitude decay exponentially. Switch to damping = 0 and the oscillation persists forever — the ideal undamped case. Real strings decay in about 1–10 seconds depending on material.

5. Higher Harmonics

Select n=6 or n=8. Toggle the envelope to see the sinusoidal shape of the maximum displacement. Note how the frequency is exactly 6× or 8× the fundamental. This integer multiple relationship is what makes musical tones sound harmonic.

6. Heavy vs. Light String

Fix T=2 N, L=1.0 m, n=1. Compare μ=0.001 kg/m (light) vs μ=0.1 kg/m (heavy). The light string vibrates ~10× faster. A cello's thicker strings give lower pitch precisely because μ is larger.
