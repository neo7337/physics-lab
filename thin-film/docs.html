Thin-Film Interference

Thin films — soap bubbles, oil slicks, anti-reflective coatings, butterfly wings — produce vivid colors through interference between light reflected from the film's top and bottom surfaces.

The color seen depends on the film thickness, the refractive index, and the angle of incidence. Constructive interference brightens certain wavelengths; destructive interference removes others.

Historical Background

1665
Robert Hooke observed colors in thin films of mica and soap bubbles, noting they depended on film thickness.
1672
Isaac Newton studied "Newton's rings" — circular interference fringes between a flat glass plate and a curved lens — and measured thin-film colors precisely.
1801
Thomas Young explained Newton's observations using his wave theory: two reflections separated by an optical path difference produce interference.
1870s
Lord Rayleigh derived the theory of anti-reflection coatings: a quarter-wave layer with n = √(n_glass) eliminates reflection entirely.
1940s
Anti-reflection coatings became standard in cameras, binoculars, and microscopes during and after WWII.

Physics

When light hits a thin film, part is reflected at the top surface (ray R₁) and part enters the film, travels to the bottom, and reflects back out (ray R₂). These two rays interfere.

The optical path difference between R₁ and R₂ is:

OPD = 2 · n · t · cos(θₜ)

where θₜ is the refraction angle inside the film (from Snell's law: n₀ sinθᵢ = n sinθₜ).

The interference depends on the total phase difference, which includes the OPD and any phase shifts from reflection.

Phase Shifts at Interfaces

When light reflects from an interface where it goes from a lower to higher refractive index, it acquires a phase shift of π (half-wavelength). No phase shift occurs for higher→lower reflection.

This simulator automatically detects the phase shift scenario and updates the constructive condition formula accordingly.

Constructive & Destructive Conditions

One Phase Flip (most common: air / glass film / glass substrate)

Constructive: 2nt cosθₜ = (m + ½)λ, m = 0, 1, 2, …
Destructive: 2nt cosθₜ = mλ, m = 1, 2, …

Two Phase Flips (or zero)

Constructive: 2nt cosθₜ = mλ, m = 1, 2, …
Destructive: 2nt cosθₜ = (m + ½)λ, m = 0, 1, 2, …

Quarter-Wave Anti-Reflection Condition

For a single-layer AR coating with one phase flip, minimum reflectance occurs at:

t = λ/(4n) → OPD = λ/2 → R = 0 when n = √(n_glass)

Reflectance Spectrum

The simulator plots R(λ) across the full visible spectrum (380–750 nm) simultaneously. Each wavelength has a different OPD/λ ratio, so different wavelengths experience constructive or destructive interference — producing the characteristic colored pattern of soap bubbles and oil films.

The reflectance formula used is the two-beam approximation (valid when R ≪ 1 at each surface):

R = (r₁² + r₂² + 2r₁r₂ cos δ) / (1 + r₁²r₂² + 2r₁r₂ cos δ)

where r₁, r₂ are the Fresnel amplitude reflection coefficients at each surface and δ = 2πOPD/λ.

Guided Experiments

1. Anti-Reflection Coating

Set n = 1.38 (MgF₂), n₂ = 1.52 (glass), λ = 550 nm. Find the quarter-wave thickness: t = λ/(4n) ≈ 100 nm. Set t = 100 nm and observe R drops to near zero for green light.

2. Soap Bubble Colors

Set n = 1.33, n₂ = 1.33 (both water/air). Vary t from 0 to 1000 nm. Watch the spectrum oscillate between different colored peaks — mimicking what you see on a soap bubble with thickness variation.

3. Oil-on-Water

Set n = 1.47 (oil), n₂ = 1.33 (water). Note only one π phase flip occurs (top surface: air→oil). Sweep t through 200–800 nm and observe the shifting rainbow of reflected colors.

4. Angle Dependence

Set a fixed thickness t = 400 nm, n = 1.5. Increase θ from 0° to 70°. The spectrum shifts to shorter wavelengths (Bragg blueshift) — this is why soap bubble colors change as you tilt them.

5. Zero Thickness (Black Film)

Set t = 0 nm. With one phase flip: the OPD = 0, which satisfies destructive condition 2nt = 0 = 0·λ, giving R ≈ 0. This is the "black soap film" — no reflection at all near t = 0.

6. High-Index Film (TiO₂)

Set n = 2.35 (titanium dioxide), t = 60 nm. Quarter-wave at λ ≈ 564 nm. Compare the much higher reflectance range vs. MgF₂ coating. High-n films give stronger interference effects.
