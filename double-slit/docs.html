Overview

Young's Double Slit experiment is one of the most important experiments in the history of physics. It demonstrates conclusively that light behaves as a wave, producing an interference pattern — alternating bright and dark bands — when passed through two narrow slits. The same effect is observed with electrons, neutrons, and even large molecules, making it a cornerstone demonstration of wave–particle duality.

Key insight: When two waves meet, they either reinforce each other (constructive interference → bright fringe) or cancel each other out (destructive interference → dark fringe). The pattern you see on the screen is a permanent record of where each type of interference occurs.

Historical Background

In 1801, Thomas Young performed his famous experiment by shining sunlight through a pinhole and then through two closely spaced slits onto a screen. The resulting alternating light and dark bands were inexplicable under Newton's corpuscular (particle) theory of light, but were perfectly predicted by Huygens' wave theory.

1678
Huygens proposes wave theory of light; every point on a wavefront is a source of secondary wavelets.
1704
Newton publishes Opticks, arguing light is a stream of particles (corpuscles).
1801
Thomas Young demonstrates the double-slit interference pattern, reviving the wave theory.
1905
Einstein explains the photoelectric effect using photons — light also has particle properties.
1927
Davisson & Germer demonstrate electrons produce the same interference pattern — wave–particle duality confirmed.
1999
C₆₀ fullerene molecules (60 carbon atoms) show double-slit interference — quantum weirdness at a macro scale.

Wave Nature of Light

A wave is a periodic disturbance that carries energy through space. For light, this disturbance is oscillating electric and magnetic fields. Every wave has:

λ
Wavelength
Distance between successive crests. Visible light: 380–750 nm. Determines color.
f
Frequency
Number of oscillations per second (Hz). Related to energy: E = hf.
A
Amplitude
Peak displacement of the wave. Intensity ∝ A².
φ
Phase
Position within the oscillation cycle (0–2π radians). Phase difference between two waves determines whether they add or cancel.

The key property enabling the double-slit effect is coherence: both slits must be illuminated by light of the same wavelength and a stable phase relationship. In practice this is achieved by using a single monochromatic source (e.g. a laser) placed far away, or by passing light through a single slit first.

Path Difference

Consider a point P on the screen at height y from the centre. Light from Slit 1 travels a distance r₁ to P, and light from Slit 2 travels r₂. The path difference is:

Δ = r₂ − r₁

For the geometry where L ≫ d (screen is far away — the Fraunhofer or far-field approximation), this simplifies to:

Δ = d · sin θ
where θ is the angle from the central axis to point P

For small angles (θ ≪ 1 rad, i.e. y ≪ L), we can further approximate:

sin θ ≈ tan θ = y / L
⟹   Δ ≈ d · y / L
In the simulator: the exact formula sin θ = y / √(y² + L²) is used (no small-angle approximation), so the pattern is accurate even for large angles.

Interference Conditions

The two waves arrive at P with a phase difference proportional to the path difference. Whether they add (constructive) or cancel (destructive) depends on how many whole wavelengths fit into Δ.

✦ Constructive (Bright Fringe)

Waves arrive in phase — crests meet crests.

Δ = mλ   (m = 0, ±1, ±2, …)
ym = mλL / d

m is the fringe order. m=0 is the central maximum.

✦ Destructive (Dark Fringe)

Waves arrive out of phase — crests meet troughs.

Δ = (m + ½)λ   (m = 0, ±1, …)
ym = (m + ½)λL / d

Halfway between bright fringes.

Intensity Formula

The total amplitude at a point is the sum of the two wave amplitudes. Using trigonometric addition, the intensity (proportional to amplitude squared) works out to:

I(θ) = 4I₀ · cos²( πd sin θ / λ )
where I₀ is the intensity from a single slit

This is the pure double-slit result. Because cos²(x) oscillates between 0 and 1, the intensity ranges from 0 (dark fringe, destructive) to 4I₀ (bright fringe, constructive — four times a single slit, because amplitude doubles and I ∝ A²).

Step-by-step derivation

  1. Wave from slit 1 at point P:   E₁ = A sin(kr₁ − ωt)
  2. Wave from slit 2 at point P:   E₂ = A sin(kr₂ − ωt)
  3. Total field: E = E₁ + E₂. Using sum-to-product: E = 2A cos(kΔ/2) · sin(k(r₁+r₂)/2 − ωt)
  4. Intensity (time-averaged) ∝ amplitude²: I = 4A² cos²(kΔ/2) = 4I₀ cos²(πΔ/λ)
In the simulator: the intensity function at main.js:72 computes sinTheta = y / √(y² + L²), then delta = d · sinTheta, then I = A² · cos²(π·delta/λ) · sinc²(...). The amplitude slider multiplies A directly.

Single-Slit Envelope

Each slit has a finite width a. A single finite slit produces its own diffraction pattern — a broad central peak with weaker side lobes. This single-slit envelope multiplies the double-slit pattern and modulates which fringes are visible.

I(θ) = I₀ · sinc²( πa sin θ / λ ) · cos²( πd sin θ / λ )
sinc(x) = sin(x)/x  |  sinc(0) = 1

The sinc² factor goes to zero when a sin θ = mλ, which creates missing orders — fringe maxima that are suppressed because they fall exactly at a single-slit minimum.

Try it: In the simulator, set slit width a very close to the slit separation d. You'll see every other fringe disappear (the m=±1 maxima of the double-slit pattern coincide with the first minimum of the single-slit envelope).

Fringe Spacing

The distance between adjacent bright fringes (fringe spacing β) is one of the most practically useful results of the experiment:

β = λL / d

This tells us immediately how each parameter affects the pattern:

ParameterIncrease itFringe spacing β
λ (wavelength) longer wavelength (redder light) ↑ increases (wider fringes)
L (screen distance) move screen farther away ↑ increases (wider fringes)
d (slit separation) move slits further apart ↓ decreases (closer fringes)
Practical application: This formula is used to measure the wavelength of light. Rearrange: λ = βd / L. Measure the fringe spacing β with a ruler, and you can calculate λ to high accuracy.

The simulator also applies an intensity rolloff at the UV (380–420 nm) and far-red (700–750 nm) ends of the spectrum, because the human eye's sensitivity drops sharply at those extremes.

White light: Real sunlight contains all wavelengths. Each wavelength produces its own fringe pattern at a different spacing — the patterns overlap, washing out higher-order fringes and giving each fringe a rainbow tinge. The central fringe (m=0, Δ=0 for all λ) stays white.

How the Simulator Works

The simulator uses the HTML5 Canvas API with requestAnimationFrame for real-time rendering. Everything runs in the browser — no server, no dependencies.

Wave Propagation Canvas

The wave field is computed per-pixel every frame using direct superposition of two sinusoidal point sources (Huygens' principle): 

for each pixel (px, py):
  r1 = distance from pixel to Slit 1    // metres
  r2 = distance from pixel to Slit 2

  w1 = A · sin(k·r1 − phase) / √r1     // 2D amplitude decay: 1/√r
  w2 = A · sin(k·r2 − phase) / √r2

  total = (w1 + w2) / 2                 // superposition
  brightness = (total + 1) / 2          // map [-1,1] → [0,1]
  pixel color = wavelength_color × brightness

The phase variable advances each frame: phase += dt × animSpeed × 4.0, giving the illusion of waves travelling outward from the slits. The factor 4.0 is a visual speed scaling (true optical frequencies of ~5×10¹⁴ Hz are imperceptible).

Screen Strip

The vertical strip on the right of the wave canvas is rendered independently using the exact intensity formula (not the wave simulation), giving a clean, accurate fringe pattern regardless of the wave animation zoom level.

Intensity Graph

The graph below the wave canvas plots I(y) by sampling the intensity function at each horizontal pixel, converting pixel position to physical screen coordinate y using the fringe spacing β to set the axis range: 

yRange = max(β × 5, 0.1 mm)             // show ±5 fringe spacings
for i in 0..plotWidth:
  y_phys = -yRange + i × (2·yRange / plotWidth)
  I      = intensity(y_phys, λ, d, a, L, A)
  plot pixel at height I × plotHeight

Coordinate System

Canvas pixels → Physical metres:
  scaleX = L / (canvasWidth − barrierX)     // metres per pixel (horizontal)
  scaleY = L × 0.8 / canvasHeight           // metres per pixel (vertical)

Slit positions (pixels):
  slit1y = centreY − (d/2) / scaleY
  slit2y = centreY + (d/2) / scaleY

State Object

All simulation parameters live in a single state object:

const state = {
  lambda:     550e-9,   // wavelength (metres)
  d:          1e-3,     // slit separation (metres)
  a:          0.1e-3,   // slit width (metres)
  L:          1.0,      // screen distance (metres)
  amplitude:  1.0,      // wave amplitude multiplier
  animSpeed:  1.0,      // animation speed multiplier
  phase:      0,        // running phase (radians)
};

Sliders update state in real time via input event listeners wired in wireControls(). Every frame the render functions read directly from state, so the simulation updates instantly.

Try These Experiments

01

Measure fringe spacing

Set λ=550 nm, d=1 mm, L=1 m. The formula predicts β = 0.55 mm. Observe the intensity graph and confirm the peak-to-peak spacing matches. Now double L to 2 m — does β double as expected?

02

Single-slit vs. double-slit

Slowly increase slit width a from 0.01 mm toward d (the slit separation). Watch the single-slit envelope narrow and swallow outer fringes. At a = d/2, the m=±2 fringes disappear (missing order).

03

Violet vs. Red light

Compare λ=400 nm (violet) with λ=700 nm (red), keeping all else equal. Red light produces fringes ~1.75× wider than violet. This is why a white-light double-slit pattern shows rainbow-colored fringes.

04

The central maximum

The m=0 fringe is always at the centre regardless of λ, d, or L. This is because at y=0, both slits are equidistant (Δ=0), so all wavelengths constructively interfere — it would appear white with real white light.

05

Wave field structure

Pause the animation and look at the wave field. Trace the bright stripes radiating from the slits — these are lines of constructive interference. Count them: they correspond to fringe orders m=0, ±1, ±2 … on the screen.

06

Fringe count vs. slit separation

With λ=550 nm and L=1 m, increase d from 0.5 mm to 5 mm. Count how many fringes fit in the central single-slit maximum. The number should scale with d/a (the ratio of slit separation to slit width).
